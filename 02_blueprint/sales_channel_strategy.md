# Sales Channel Strategy — FitLedger

## Primary channel recommendation: **Lemon Squeezy or Gumroad** (digital storefront), with the
**Shopify store as a strong second** given the owner already has a connected Shopify store.

| Channel | Fit | Notes |
|---|---|---|
| **Lemon Squeezy** | ★★★★★ | Merchant of record (handles EU VAT/sales tax on digital goods — significant for an app sold globally); license keys built in; ~5% + 50¢ |
| **Gumroad** | ★★★★☆ | Fastest setup, discovery marketplace, 10% flat; weaker tax handling (varies) |
| **Shopify (owner's existing store)** | ★★★★☆ | Already connected via MCP in this workspace — product page + Digital Downloads app; owner controls brand; owner handles tax settings |
| Etsy | ★★☆☆☆ | Buyers expect $5–25 printables; wrong price band for a $39 app |
| App stores | ✗ for v1 | PWA distribution avoids review processes and 15–30% fees; revisit if native apps ship |

## Recommended setup (pending owner approval — nothing will be created without it)

1. **Storefront:** one product page (copy delivered in Phase 6) + checkout.
2. **Delivery:** zip download (app + docs + bonuses) + hosted-app access note. License key
   optional at v1 (honor-system personal license keeps friction low; revisit if piracy hurts).
3. **Landing page:** static page (can be hosted free on Cloudflare/Netlify) with demo
   screenshots and the comparison table; links to checkout.

## Discovery / traffic (detailed in Phase 8)

Organic-first: anti-subscription angle is natively shareable on fitness TikTok/Reddit/X;
"local-first software" communities; SEO terms like "workout tracker no subscription",
"one-time purchase macro tracker", "MyFitnessPal alternative one time". Paid ads deferred
(see Phase 8 caution plan).

## Platform-rule check

Health-adjacent digital products are permitted on all candidate platforms; the app makes no
medical claims (see `legal_risk_review.md`). No platform prohibits one-time-license web apps.
