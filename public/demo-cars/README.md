# Demo car photography

The demo archive renders original blueprint **illustrations** by default (no
trademarked or copyrighted assets).

To use real licensed photography instead, drop image files here and set the
`photo` field on the matching vehicle in `lib/demo-data.ts`, e.g.:

    photo: "/demo-cars/2007-911-turbo.jpg"

Use only images you have the rights to (your own photos, CC0 / public-domain,
or properly attributed Creative Commons). Do not use official Porsche press
imagery or any image containing the Porsche crest/logo. Add a credit line to
`CREDITS.md` for any Creative Commons photo you include.
